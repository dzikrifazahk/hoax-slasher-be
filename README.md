## Run Locally

Clone the project

```bash
  git clone https://github.com/dzikrifazahk/hoax-slasher-be
```

Go to the project directory

```bash
  cd hoax-slasher-be
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run start
```

#### BEFORE START SERVER
Makesure you have .env files
Create database
Configure Database
